# Overzicht {#handling-fmrms-compatibility-overview}

Er zijn twee soorten verzoeken die gerelateerd zijn aan de compatibiliteit met Flash Media Rights Management Server 1.x. Een type aanvraag wordt gebruikt om 1.x-clients te vragen een upgrade uit te voeren naar een runtime die Adobe Access 2.0 of hoger ondersteunt. Een andere methode wordt gebruikt om 1.x-metagegevens bij te werken naar de Adobe Access-indeling voordat een licentie kan worden aangevraagd.
